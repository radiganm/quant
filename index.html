Welcome to Quant.

The Quant project provides software tools for Quantitative Finance.


## to obtain quant:
$ git clone https://github.com/radiganm/quant.git

The Quant project consists of the following modules:

Portfolio

Software libraries and tools for portfolio balancing. This currently supports Markowitz portfolio optimization.
